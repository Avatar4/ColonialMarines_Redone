19 сентября 2015

ZVe updated:

  • Разблокированы кнопки для интерфейса и создания макросов. Спец. команды для оружия, а на данный момент их только две, подсмотреть можно в специальной вкладке при наличии оружия в руках. Точное название команды идет после двоеточия - прим: "Rifle: selectfire" - в данном случае нужен будет только кусок - selectfire.
  • Добавлена возможность блокировать контроль шаттла с консолей которые снаружи.
  • Исправлен баг с riot shotgun. Была возможность вынуть магазин, когда на самом деле - этого делать нельзя.
  • Теперь можно расстреливать резину которая поверх стен и дверей.

14 сентября 2015

ZVe updated:

  • В тестовом режиме введена механика стрельбы для полностью автоматического оружия. В данный момент это м41а и м39 смг. Если включен харм - будет стандартный одиночный огонь, а в близи можно будет бить "прикладом" как и раньше собственно.
  • Изменены параметры магазинов у м37а2, м41а и м39 смг.
  • Спрайты м4а3, м37а2, м41а, м39 от WalterJe.
  • Убран пулемет "пила" из вендомата.
  • Добавлены подстволные гранаты для м41а в вендомат.
  • Теперь чтобы взять оружие в две руки - достаточно кликнуть по оружию в активной руке, а ctrl+клик убран.
  • Замена магазина происходит теперь кликом магазина по оружию - старый будет сброшен на землю, а новый вставлен. Для двуручного оружия это обязательно, другим способ поменять магазин не выйдет, а для одноручного можно и по старому - просто кликнув пустой рукой.
  • Фонарь теперь можно прикрепить к м39 и м41а. Кнопку вкл/выкл - Object > toggle gunlight. Фонарь какой попало конечно же "приклеить" не выйдет. Отверткой можно открутить, если это конечно вообще нужно.
  • Увеличен урон у дробовика.
  • Ксеноморфы вновь не могут бегать по столам как дома. Можно баррикадироваться.

ZVe updated:

3 сентября 2015

ZVe updated:

  • Для стрельбы с некоторого вида огнестрела, необходимо держать его в обеих руках. Чтобы это сделать, нужно нажать по оружию в активной руке контрол+ЛКМ.
  • Добавлен хоткий для передергивания помпы дробовика нажатием шифта+ЛКМ по любому турфу.

2 сентября 2015

ZVe updated:

  • Исправлена мелкая недоработка при использовании способностей посредством шифта+ЛКМ. Теперь ксеноморф смотрит в сторону курсора, тем самым избегая ситуации, когда "снаряд" попадает в него же, если плевок происходил в цель за спиной.
  • Портирован механ развития эмбриона с классической сборки. Теперь полно времени чтобы дойти до операционной.
  • Вернулась способность разваривания вент для ксеноморфов.
